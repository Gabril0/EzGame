# EzGame

Easy 2D browser game engine

## Prerequisites

- Docker
- Docker Compose

## How to Run

1. Clone the repository:
   ```sh
   git clone https://github.com/Gabril0/EzGame.git
   cd EzGame
   ```

2. Build and start the Docker containers:
```docker
docker-compose up --build
```
4. Open your browser and navigate to
```
http://localhost:8080
```
